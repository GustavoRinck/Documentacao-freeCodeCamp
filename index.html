Introdução

JavaScript é uma linguagem de programação interpretada usada para desenvolvimento web.

Ela permite a criação de interatividade nas páginas HTML.

Executa diretamente no navegador sem necessidade de compilação.

Possui suporte a paradigmas funcionais e orientados a objetos.

É usada tanto no frontend quanto no backend com Node.js.

console.log('Olá mundo');
  • Leve
  • Interpretada
  • Alta performance
Variáveis e Tipos

JavaScript permite a criação de variáveis com `var`, `let` e `const`.

O tipo de dado pode ser número, string, boolean, objeto, array, entre outros.

O JavaScript é fracamente tipado e dinâmico.

Exemplos de declaração:

let nome = 'João'; const idade = 30; var ativo = true;
  • string
  • number
  • boolean
  • object
  • undefined
Funções

Funções são blocos de código reutilizáveis.

Podem ser declaradas com `function`, `const` (arrow functions) e `class` (métodos).

Permitem passar argumentos e retornar valores.

function somar(a, b) { return a + b; } const subtrair = (a, b) => a - b;
  • Função nomeada
  • Função anônima
  • Arrow function
  • Função de callback
  • Função construtora
Eventos

Eventos são ações do usuário ou do sistema que podem ser capturadas.

Podem ser adicionados com `addEventListener`.

Exemplos: clique, envio de formulário, carregamento da página.

document.addEventListener('click', () => alert('Clique detectado')); btn.addEventListener('mouseover', handler);
  • click
  • submit
  • mouseover
  • keydown
  • load
DOM Manipulação

O DOM representa a estrutura HTML como um objeto JS.

Você pode acessar e modificar elementos HTML usando JS.

Seletores comuns: `getElementById`, `querySelector`.

document.getElementById('titulo').innerText = 'Novo título'; document.querySelector('.classe').classList.add('ativo');
  • Manipulação de conteúdo
  • Manipulação de classes
  • Criação de elementos
  • Remoção de elementos
  • Navegação entre nós
